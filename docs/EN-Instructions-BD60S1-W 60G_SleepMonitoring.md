60G Sleep Monitoring Radar ST-BD60S1-WT/ST-BD60S1-Z 

This 60GHz millimeter wave sleep monitoring radar adopts FMCW radar technology to detect human body micro-movement, breathing and heartbeat without contact. It monitors real-time in-bed status, identifies awake, light sleep, deep sleep and abnormal struggling states, and automatically generates sleep quality reports after continuous monitoring. 

## **Key Advantages** 

- Detection is not affected by light, dust, noise, airflow, temperature or humidity 

- Non-contact monitoring, no wearable devices required 

- Two hardware versions available: 

- ST-BD60S1-WT: 2.4GHz Wi-Fi model (standalone use) 

- ST-BD60S1-Z: Zigbee model (requires Tuya Zigbee Gateway) 

- Wall-mounted inclined installation for bedside sleep monitoring 

## **2. Core Functions** 

1. Sleep Quality Analysis Record in-bed/out-of-bed status, track sleep duration, distinguish light sleep / deep sleep / awake status, generate objective sleep score and historical records. 

2. Respiratory Rate Monitoring Real-time detection of breathing cycles, record average respiratory rate during sleep. 

3. Heart Rate Monitoring Capture subtle heartbeat vibration, output real-time and average heart rate data. 

4. Abnormal Struggle Alert Trigger reminder when continuous large body movement (dyspnea, physical discomfort) is detected. 

5. Human Presence & Out-of-Bed Record Track real-time presence, record out-of-bed times and roll-over counts overnight. 

6. Local Timing Statistics Customize unmanned delay timing for scene linkage.

## **4.1 Standard Installation Steps** 

1. Slide the front panel upward to detach the mounting base from the main unit. 

2. Align the base against the wall, mark drilling positions, drill holes and insert plastic expansion tubes. 

3. Secure the base with fixing screws, then cover screw caps for aesthetics. 

4. Attach the radar host back to the base, connect Type-C power cord and complete network pairing via Tuya App. 

## **4.2 Critical Installation Requirements (Must Follow for Accurate Detection)** 

1. Mount the radar at a **45° downward inclined angle** facing the bed surface. 

2. Recommended installation height: 1 m above the mattress surface. 

3. Effective detection distance zones: 

   - L1 (Respiration & Heartbeat): Max 1.5 m 

   - L2 (Sleep Posture Detection): Max 2.5 m 

   - L3 (Human Presence Detection): Max 2.5 m 

   - Horizontal & Vertical beam angle: 40 ° × 40 ° 

Keep the radar detection window unobstructed; avoid metal bed frames, thick curtains, ceiling fans, air conditioners and vibrating equipment within coverage. 

## **4.3 Installation Warnings** 

- Do not install flat on ceiling or vertically facing the wall; detection accuracy will fail. 

- Avoid drilling through wall electrical wires or water pipes during installation. 

- This device supports single-person monitoring only; multiple people in bed will cause data misjudgment.

## **8. Sleep Report Generation Rules** 

1. The device must be switched to Sleep Detection Mode to generate sleep reports. 

2. Valid monitoring duration requirement: 4 hours ≤continuous in-bed time ≤12 hours. 

3. Report trigger condition: After the user leaves the detection area and maintains "Nobody" status for 5 consecutive minutes, the system automatically calculates and saves the sleep report. 

4. If monitoring time exceeds 12 hours, the system will split data into multiple daily reports.
